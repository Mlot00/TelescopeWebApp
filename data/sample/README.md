# Sample data placeholder

Ten katalog jest miejscem na mały fixture DL3 do smoke testów.

Repozytorium zawiera minimalny fixture demo w `sample/crab_dl3/` z plikami
`hdu-index.fits.gz` i `obs-index.fits.gz`, żeby podstawowa walidacja datasetu
przechodziła po świeżym klonie i uruchomieniu Dockera.

Wymagane pliki dla podstawowej walidacji:
- `hdu-index.fits.gz`
- `obs-index.fits.gz`

Nie commituj pełnych danych produkcyjnych do repozytorium.
